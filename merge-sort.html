📘 演算法說明

+

合併排序是一種經典的分治法排序演算法，先把陣列一直切成左右兩半，直到每段只剩 1 個元素，再依序把小段合併成大段。

+
    +
  • 先分割：把大問題拆成多個小問題。
    • +
  • 再合併：比較左右子陣列的最前面元素，依序放回原陣列。
    • +
  • 適合處理大量資料，時間效率通常優於 O(n²) 的簡單排序法。
    • +
+
+ 提示：這個頁面會把「切分範圍」與「合併結果」拆成多個步驟，方便觀察分治法怎麼逐步完成排序。 +

🪜 合併排序流程

+
    +
  1. 分割陣列：把區間 `[left, right]` 持續切成左右兩半。
    2. +
  2. 遞迴到底：當區間只剩一個元素時，它本身就已經有序。
    3. +
  3. 比較左右子陣列：將較小值依序放回原陣列。
    4. +
  4. 補上剩餘元素：若某一邊先取完，就把另一邊剩下的值接到後面。
    5. +
  5. 完成排序：所有子問題合併後，整個陣列就會有序。
    6. +
