📚 演算法概念

合併排序法採用了分而治之 (Divide and Conquer) 的策略：

  • 分割 (Divide)：將陣列從中間切半，不斷遞迴切割，直到每個子陣列只剩 1 個元素 (視為已排序)。
  • 克服 (Conquer)：無須特別處理，因為單個元素本身就是排序好的。
  • 合併 (Merge)：將兩個已排序的子陣列，透過比較大小，合併成一個新的且排序好的陣列。
💡 核心特點： 無論資料初始狀態如何，時間複雜度都非常穩定。但缺點是需要額外的記憶體空間來暫存合併的結果。

⏱️ 複雜度分析

時間複雜度
O(n log n)
空間複雜度
O(n)

💻 程式碼對應 (In-place 索引版)

function mergeSort(arr, left, right) { if (left >= right) return; let mid = Math.floor((left + right) / 2); mergeSort(arr, left, mid); mergeSort(arr, mid + 1, right); merge(arr, left, mid, right); } function merge(arr, left, mid, right) { let temp = [], i = left, j = mid + 1; while (i <= mid && j <= right) { if (arr[i] <= arr[j]) temp.push(arr[i++]); else temp.push(arr[j++]); } while (i <= mid) temp.push(arr[i++]); while (j <= right) temp.push(arr[j++]); for (let k = 0; k < temp.length; k++) { arr[left + k] = temp[k]; } }

void mergeSort(vector<int>& arr, int left, int right) { if (left >= right) return; int mid = left + (right - left) / 2; mergeSort(arr, left, mid); mergeSort(arr, mid + 1, right); merge(arr, left, mid, right); } void merge(vector<int>& arr, int left, int mid, int right) { vector<int> temp; int i = left, j = mid + 1; while (i <= mid && j <= right) { if (arr[i] <= arr[j]) temp.push_back(arr[i++]); else temp.push_back(arr[j++]); } while (i <= mid) temp.push_back(arr[i++]); while (j <= right) temp.push_back(arr[j++]); for (int k = 0; k < temp.size(); k++) { arr[left + k] = temp[k]; } }

📝 演算法步驟說明

  1. 遞迴分割: 找到陣列中間點 `mid`，將陣列切分為左半 `[left, mid]` 與右半 `[mid+1, right]`。
  2. 觸底反彈: 當子陣列長度為 1 時，停止分割並開始返回。
  3. 建立暫存: 在合併階段，雙指針 `i`, `j` 分別指向左右子陣列的開頭，互相比較大小。
  4. 依序放入: 將較小的值放入 `temp` 暫存陣列，並移動對應的指針。
  5. 寫回原陣列: 比較完畢後，將剩下的元素全數補上，最後將 `temp` 陣列的結果寫回原陣列對應的區間。
