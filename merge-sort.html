合併排序 (Merge Sort)

比照 `select-sort.html` 的互動模式，逐步展示分割與合併的排序過程。

互動示範

步驟 0 / 0

演算法說明

合併排序採用分治法，先把陣列切成左右兩半，直到每段只剩一個元素，再把小段依序合併成有序結果。

  • 先分割陣列。
  • 遞迴排序左右子陣列。
  • 比較左右兩邊的最前端元素並合併。

複雜度

時間複雜度
O(n log n)
空間複雜度
O(n)

程式碼範例

function mergeSort(arr, left = 0, right = arr.length - 1) { if (left >= right) return; const mid = Math.floor((left + right) / 2); mergeSort(arr, left, mid); mergeSort(arr, mid + 1, right); merge(arr, left, mid, right); } function merge(arr, left, mid, right) { const leftPart = arr.slice(left, mid + 1); const rightPart = arr.slice(mid + 1, right + 1); let i = 0, j = 0, k = left; while (i < leftPart.length && j < rightPart.length) { arr[k++] = leftPart[i] <= rightPart[j] ? leftPart[i++] : rightPart[j++]; } while (i < leftPart.length) arr[k++] = leftPart[i++]; while (j < rightPart.length) arr[k++] = rightPart[j++]; }

流程摘要

  1. 將區間 `[left, right]` 持續切成左右兩半。
  2. 當區間只剩一個元素時停止切分。
  3. 回溯時比較左右子陣列，逐步寫回原陣列。
  4. 最後整個陣列完成排序。
