插入排序 (Insertion Sort)

交互式步驟演示與程式碼對應

交互式演示

第 0 / 0 步

演算法概念

插入排序建構已排序序列，對於每個元素將其插入已排序區間的正確位置。

  • 從第二個元素開始，將目前元素（key）與前面已排序元素比較
  • 若前一個元素較大則右移，直到找到適當插入位置
  • 重複此過程，直到整個陣列排序完成
核心特點： 插入排序為穩定排序，對近乎有序資料特別高效。

複雜度分析

時間複雜度
O(n)
空間複雜度
O(1)

程式碼對應

function insertionSort(arr) { for (let i = 1; i < arr.length; i++) { let key = arr[i]; let j = i - 1; while (j >= 0 && arr[j] > key) { arr[j + 1] = arr[j]; j--; } arr[j + 1] = key; } return arr; }

演算法步驟說明

  1. 外層循環 (i): 從索引 1 開始，把當前元素視為 key
  2. 內層比較 (j): 向已排序區段從右至左比較並右移元素
  3. 插入步驟: 將 key 放置在合適位置 arr[j + 1]
  4. 完成排序: 重複直到整個陣列完成排序
